Privacy Policy

1. Information Collection

Our Bot collects the following information to provide and improve services:

  • Discord Server IDs and configuration settings
  • User IDs for command usage and permissions
  • Command usage statistics and patterns
  • Server activity data for moderation purposes
  • Custom configurations set by server administrators

We only collect information necessary for Bot functionality and server management.

2. Use of Information

Collected information is used for:

  • Providing and maintaining Bot services
  • Implementing server-specific configurations
  • Analyzing usage patterns to improve features
  • Preventing abuse and maintaining security
  • Supporting server moderation efforts

We never sell or share your data with third parties for marketing purposes.

3. Data Storage and Security

We implement industry-standard security measures:

  • Encrypted data storage and transmission
  • Regular security audits and updates
  • Limited access to data by authorized personnel
  • Automatic data cleanup for inactive servers

4. User Rights and Control

You have the right to:

  • Request access to your stored data
  • Request deletion of your data
  • Opt-out of specific data collection features
  • Receive explanations about data usage

Contact our support team to exercise these rights.

5. Data Retention

We retain data as follows:

  • Active server data: Maintained while Bot is in server
  • Command logs: 30 days
  • Configuration settings: Until changed or Bot removal
  • Inactive server data: Automatically deleted after 90 days
